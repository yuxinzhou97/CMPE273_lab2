# CMPE273_lab2

## Instructions
* See requirements: https://github.com/sithu/cmpe273-fall20/tree/master/lab2
* Install dependencies:
```
pipenv install pyzmq
```
* Open three terminals, run in order:
* Terminal 1:
```
pipenv shell
python dashboard.py
```
* Terminal 2:
```
pipenv shell
python worker.py
```
* Terminal 3:
```
pipenv shell
python generator.py
```
